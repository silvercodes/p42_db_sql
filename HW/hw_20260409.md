## Задача 1

Вывести список сотрудников из **IT-отдела** (`department_id = 3`), у которых зарплата **больше 1500**.

| Столбец | Описание |
|---------|----------|
| `last_name` | Фамилия |
| `first_name` | Имя |
| `salary` | Зарплата |
| `evaluation` | Оценка через `CASE`: `>= 2000` → `'high'`, `>= 1600` → `'medium'`, иначе → `'low'` |

Отсортировать по `salary` по убыванию.
